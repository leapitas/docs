# Installation

Enkel EHF is available as an app from Microsoft Marketplace, also known as Appsource. Install it directly from within Business Central or from the Microsoft Marketplace website.

## Installation from Business Central

1. In Business Central, search for **Extension Management**
2. Click **Appsource Gallery** 
3. Search for **Enkel EHF** and click it.
4. Select **Install App** to start the installation
5. Select Language and then click **Install**

## Installation from AppSource

1. Visit the [Enkel EHF listing on Microsoft AppSource](https://marketplace.microsoft.com/en-us/product/PUBID.leapitas1641658239712%7CAID.enkel-ehf%7CPAPPID.a5654876-da09-40fd-a0f3-8aba4a9fc424?tab=Overview)
2. Click **Free trial**
3. Follow the instructions to install the app in your Business Central environment

The app will automatically install.
