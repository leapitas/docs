# Customer Document Sending Profiles

To send EHF documents to a customer, the EHF document sending profile must be assigned on the customer card.

## Assigning the Profile to a Customer

1. Search for **Customers** and open the customer card for the customer you want to set up
2. Navigate to the **Shipping and Billing** section (or use the search field on the card)
3. In the **Document Sending Profile** field, select the EHF profile (e.g. "EHF")

Once assigned, all supported documents sent to this customer will be processed as EHF documents through the PEPPOL network.

## Bulk Assignment

You can also assign the document sending profile to multiple customers at once:

1. Search for **Customers** to open the customer list
2. Select the customers you want to assign the profile to
3. Choose **Actions** > **Documents** > **Document Sending Profile**
4. Select the EHF profile and confirm

## Automatic Assignment via PEPPOL Lookup

If you have enabled the PEPPOL Directory lookup feature, the document sending profile can be automatically assigned to customers who are registered in the PEPPOL Directory. See [PEPPOL Profile Update](peppol-profile-update.md) for more details.
