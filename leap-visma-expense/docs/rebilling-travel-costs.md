# Re-billing of Travel Costs

Re-billing of travel costs is supported through standard functionality in Business Central and is set up per project.

1. On project card under tab "Prices and discounts" click on Finance Account button.
2. Optionally select a project task if you want re-billing to only occur on certain project tasks.
3. Specify the finance account to be re-billed in field Finance Account No.
4. Specify unit cost factor. A unit cost factor of 1 means that you re-bill the cost without markup.
5. Ensure that the project has specified a project posting group under Posting on the project card.
6. If you specify a value in the field Sales Expense Account (contract) on the project posting group, the finance account used on the invoice will be converted to this value. If not, the finance account that was used when posting the project journal is used.
