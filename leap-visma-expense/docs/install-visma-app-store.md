# Install in Visma App Store

You will receive an invitation code from us that should be entered in Visma App Store.

Go to apps.visma.com and log in with the same login you use in Visma.net (Alternatively go to Visma Home and click on App Store).

Click the "Use invitation code" button and enter the invitation code you received.

You will now see what types of data this application wants to retrieve from Visma Expense.

By clicking Integrate, you approve that the application gets access to the relevant data. This is a prerequisite for reading data from Visma Expense to Business Central.
