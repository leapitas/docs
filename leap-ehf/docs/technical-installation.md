# Technical Installation

Requires three apps that must be installed in the customer's environment. These apps will be provided upon ordering the product.

1. **Leap Logiq Api Client** - The app contains the communication layer necessary to communicate with the message center
2. **Leap E-Document Format Peppol Bis 3.0 (Attachment)** - Adds EHF 3.0 with attachment as a new option for e-document services
3. **Leap E-Document Service Logiq Api** - Adds Logiq Api as a new option for e-document services
4. **Leap Peppol Lookup** - (Optional) Allows you to automatically update the document sending profile on customers based on whether they are registered in the Peppol Directory

Leap Logiq Api Client must be installed first. After that, the order in which the other apps are installed is arbitrary.
