# Related Setup

## Units of Measure

**International Standard Code**

Check that units of measure you use have a value filled in the "**International Standard Code**" field.

For a full list of possible values see [UNECE Recommendation 20](https://docs.peppol.eu/pracc/catalogue/1.0/codelist/UNECERec20/).

## Countries/Regions

Check that countries you want to send EHF documents to have a value in the **VAT Scheme** field. For Norway `0192` should be used.

For a full list of possible values see [PEPPOL Participant identifier schemes](https://docs.peppol.eu/edelivery/codelists/old/v8.0/Peppol%20Code%20Lists%20-%20Participant%20identifier%20schemes%20v8.0.html).

## VAT Posting Setup

Check that the VAT configurations you use have a value filled in the **Tax Category** field.

For a full list of possible values see [UNCL5305 Tax Category](https://docs.peppol.eu/pracc/catalogue/1.0/codelist/UNCL5305/).
