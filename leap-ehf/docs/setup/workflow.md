# Workflow Setup

1. Search for "**Workflows**" in the search menu and open the corresponding link
2. Click the "**New workflow from template**" button
3. In the window that appears, select "**Send to one service**" under E-Document

:::info If you don't see the option "Send to one service"
Search for "**Workflow Templates**" in the search menu and open the corresponding link.
Under Actions select "**Reset Microsoft Templates**".
The option should now appear in the list.
:::

## Configure the workflow

1. Click into the workflow that was created
2. Give the workflow a **Code** e.g. "EHF"
3. Give the workflow a **Description** e.g. "EHF with attachment"
4. On the row below **Workflow Steps**, click on the cell in the "**Then Response**" column
5. In the window that appears, in the "**E-Document Service**" field select the e-document service you created earlier. Then click OK
6. Check the **Enabled** checkbox on the workflow
