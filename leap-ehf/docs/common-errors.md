# Common Error Messages

## Exchange not found

This error message usually indicates that the recipient of the document is not registered in the Peppol Directory. The document must then be sent via another channel such as email.

**Solution**:
- Check that the customer's organization number and country/region is correctly registered
- Verify that the customer is registered in the Peppol Directory
- Alternatively, use a different document sending profile (e.g. email) for this customer
