# About Leap EHF

Leap EHF is an application for Microsoft Dynamics 365 Business Central that provides your business with a complete solution for electronic invoicing (EHF) via the PEPPOL network.

Full documentation available in Norwegian only.

## Key Features

- **PEPPOL 3.0 Compliance**: Full support for PEPPOL 3.0 standard
- **Status and Receipts**: Real-time tracking of sent messages
- **Multi-Channel Distribution**: Invoice distribution across various channels
- **Business Central Integration**: Fully integrated with Business Central's standard e-document functionality
- **PEPPOL Directory Lookup**: Verify recipients are connected to the PEPPOL network
- **PDF Attachments**: Send invoices with PDF attachments included
- **Additional Attachments**: Add timesheets, contracts, or other documents as supplementary attachments

For complete documentation, please switch to Norwegian (Norsk) using the language selector.
