# Import employees and resources to Business Central
When the setup is completed, you can start importing employees and resources from Visma Employees to Business Central.

## Starting the import

### 1. Open the employee list
Navigate to the employee list in Business Central.

### 2. Choose import type
Select **Actions** and **Visma**. You have two options:

#### Update all employees
- **Updates and creates** employees based on changes in Visma
- **Processes all employees** who have changes since the last import
- **Also creates resources** if specified in the setup

#### Update employee
- **Updates only** the employee you have selected
- **Useful for** testing the import or updating specific employees
