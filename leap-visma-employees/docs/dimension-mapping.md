# Mapping of dimensions
If there are differences or missing dimensions between Business Central and Visma Employees, this can be mapped in a separate setup.

## When do you need dimension mapping?
Dimension mapping is necessary when **Dimension codes** are different between the systems.

## Open dimension mapping

### Via the setup
1. **Search for** and open **"Setup for Visma employee import"**
2. Click on **"Dimension mapping"** to open the screen for dimension mappings

## Setting up mappings

| Field | Description |
|-------|-------------|
| **Visma dimension** | The dimension code that comes from Visma Employees |
| **Dimension** | Corresponding dimension code in Business Central |
