# Create a General Signing Document
Leap Sign facilitates the electronic signing of documents within Business Central. This signing process can relate to various master data entities such as customers, vendors, contacts, and employees.

The solution also supports creation of entirely independent signing documents that are not linked to any master data whatsoever.

## Create a Signing Document from Master Data
Leap Sign supports creating signing documents directly from customer, vendors, contacts and employees. When this is done, the created signing document is automatically filled in with details from the related entity.

### From Customer and Vendor
For customer and vendor the action is located at:
**New Document → New Signing Document**

### From Contact and Employee
For contact and employee the action is located at:
**Process → New Signing Document**

:::tip Factbox
When you access entities that have related Signing Documents, the **Signing Documents factbox** will show a list of the Signing Documents.
:::

## Create a Signing Document from Scratch

Within the **Signing Documents** list page, you have the option to manually create signing documents. There's no obligation to associate these signing documents with any interconnected data within the system.

This flexibility allows you to utilize signing documents for the electronic signing of various types of documents, regardless of whether they are linked to specific data entities.
