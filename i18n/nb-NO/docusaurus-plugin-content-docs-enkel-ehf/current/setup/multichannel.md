# Multikanal

Multikanal er en løsning som kan aktiveres slik at meldingssentralen tar over ansvaret for hvilken kanal (EHF (Peppol), E-post, eller print) som skal benyttes for kunden.

Du kan selv velge hvilke kanaler du ønsker å benytte. Sending via e-post har samme transaksjonspris som ved sending på EHF, men utskrift og sending via post koster mer. Se [leapit.no/product/ehf](https://www.leapit.no/product/ehf) for oppdaterte priser.

Tjenesten vil først forsøke å sende som EHF. Dersom kunden ikke kan motta bilag som EHF forsøkes deretter e-post. Dersom sending på e-post ikke kan utføres, som oftest fordi det ikke er lagt inn en e-postadresse på kundekortet, så forsøkes det å sendes som brev.

For å sende som brev må adresseinformasjon være fullstendig utfylt på kundekortet.

## Aktivere multikanal

1. Søk på "**Logiq oppsett**" i søkemenyen og trykk på tilhørende lenke
2. Under **Multikanal**, huk av for "**Bruk multikanal**"
3. Under **Kanaler**, velg de kanalene du ønsker at bilag skal sendes på
