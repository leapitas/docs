# Oppsett av profil for dokumenttjeneste

1. Søk etter "**Profiler for dokumentsending**" og trykk på tilhørende lenke
2. Trykk på **Ny**
3. Gi profilen en **Kode**, f.eks. "EHF"
4. Gi profilen en **Beskrivelse**, f.eks. "EHF med vedlegg"
5. Under **Sendealternativer**, i felt "**Elektronisk dokument**" velg "Utvidet e-dokumenttjenesteflyt"
6. I felt "**Flytkode for elektronisk dokumenttjeneste**" velg Arbeidsflyten du opprettet tidligere

Du kan nå legge opp kunder du ønsker skal motta EHF med denne profilen ved å legge den inn på kundekortet i felt "**Profil for dokumentsending**".
