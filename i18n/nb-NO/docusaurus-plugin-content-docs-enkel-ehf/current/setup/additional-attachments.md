# Sende med andre typer vedlegg sammen med EHF

Hvis du ønsker å sende med andre type vedlegg enn fakturakopi kan dette aktiveres på følgende måte:

1. Søk på "**Logiq oppsett**" i søkemenyen og trykk på tilhørende lenke
2. Under **Valg for Peppol format**, huk av for "**Legg til vedlegg**"

Vedlegg som er lagt inn på salgsbilaget vil nå legges ved som del av EHF-bilaget som genereres. Alle vedlegg legges til som standardvalg, men det er mulig å ekskludere spesifikke vedlegg ved å navigere til vedleggslisten.

## Støttede filtyper

Følgende filtyper kan legges til som vedlegg:
- .csv
- .pdf
- .png
- .jpeg
- .jpg
- .xlsx
- .ods
