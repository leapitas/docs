# Sende dokument på nytt

Dersom det mangler oppsett som f.eks. mva-skjema på land/områder så kan det være at EHF bilag vil feile under sending. I slike tilfeller kan man regenerere EHF-bilaget og sende på nytt etter man har korrigert oppsettet.

## Fremgangsmåte

1. Naviger til E-dokumentet som har feilet og trykk deg deretter inn på **E-dokumentkortet**
2. Trykk på "**Opprett dokument på nytt**". Dette vil regenerere EHF-en og ta hensyn til eventuelle endringer som er gjort i oppsett
3. Trykk på "**Send dokument**". Dette vil sende det nygenererte bilaget til meldingssentralen
