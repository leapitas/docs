# Oppfølging fra rollesenter

## Fliser på rollesenter

Det er tilgjengeliggjort fliser på rollesenter. Rollesentrene som har dette tilgjengelig er:
- Forretningsleder
- Kundeansvarlig
- Ordrebehandler

Du kan her se fliser som summerer opp antall e-dokumenter som er behandlet og feilet denne månedet. Du kan også se antall e-dokumenter som er pågående.

## Marker e-dokumenter som manuelt sjekket

Fra **E-dokumenter** kan du markere e-dokumenter som manuelt sjekket for å skjule dem fra flisene på rollesenteret. Dette kan være nyttig dersom e-bilagene feilet fordi mottakeren ikke er registrert i Peppol-registeret.

Da vil bilaget bli liggende som feilet uten noen måte å slette e-dokumentet. Trykk på knappen "**Angi manuelt sjekket**" i båndet for å merke bilagene. Du kan også trykke på knappen på nytt for å fjerne merkingen igjen.
