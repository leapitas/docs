# Teknisk installasjon

Krever tre apper som må installeres på kundens miljø. Disse appene vil bli tilsendt ved bestilling av produktet.

1. **Leap Logiq Api Client** - Appen inneholder kommunkasjonslaget som er nødvendig for å kommunisere med meldingssentralen
2. **Leap E-Document Format Peppol Bis 3.0 (Attachment)** - Legger til EHF 3.0 med vedlegg som et nytt valg for e-dokumenttjenester
3. **Leap E-Document Service Logiq Api** - Legger til Logiq Api som et nytt valg for e-dokumenttjenester
4. **Leap Peppol Lookup** - (Valgfri) Lar deg oppdatere profil for dokumentsending automatisk på kunder basert på om de er registrert i Peppol-registeret

Leap Logiq Api Client må installeres først. Deretter er det vilkårlig hvilken rekkefølge de andre appene installeres i.
