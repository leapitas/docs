# Varekonfigurasjon

For nøyaktige varespesifikasjoner som sendes til Freja, sørg for at følgende felt er fylt ut på hver vare i Business Central:

| Felt | Plassering i Business Central |
|------|-------------------------------|
| **Nettovekt** | Varekort |
| **Bruttovekt** | Varekort (brukes som reserve hvis Nettovekt er tom) |
| **Enhetsvolum** | Varekort |
| **Høyde / Bredde / Dybde** | Vare-enheter (valgfritt, men anbefalt) |
| **Varesporingskode** | Varekort (kun nødvendig hvis varen spores med lotnummer, serienummer eller utløpsdato) |

Disse verdiene inkluderes i varespesifikasjonsfilen som sendes til Freja når du eksporterer varedata.
