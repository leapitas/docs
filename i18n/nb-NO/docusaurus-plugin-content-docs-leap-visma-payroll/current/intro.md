# Om Visma Payroll Import

Leap Visma Payroll Import overfører lønnstransaksjoner fra Visma Payroll til Dynamics 365 Business Central, og sikrer nøyaktig og effektiv lønnsbehandling.

## Nøkkelfunksjonalitet

- **Automatisk dataoverføring**: Automatisk import av lønnstransaksjoner fra Visma Payroll
- **Fleksibel kontotilordning**: Tilpass kontoer mellom systemene
- **Dimensjonsstøtte**: Støtte for overføring av dimensjoner og dimensjonsverdier
- **Gruppering av transaksjoner**: Reduser antall transaksjoner ved å gruppere like poster
- **Feilhåndtering**: Robust håndtering av manglende kontoer og dimensjoner
