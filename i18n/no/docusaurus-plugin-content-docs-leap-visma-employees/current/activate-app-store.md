# Aktivere App Store i Visma
Leap Visma Employees Import ligger tilgjengelig på Visma sin markedsplass "Visma App Store". Dersom du ikke har Visma App Store tilgjengelig som vist i stegene under må du kontakte Visma eller din Visma partner for å få hjelp.

## Aktivering via Visma.net Admin
### 1. Logg inn i Visma.net Admin
Start med å logge inn i Visma.net Admin med administratortilgang.
### 2. Finn riktig kunde og selskap
Finn riktig kunde og det selskapet du ønsker å bruke integrasjonen mot.
### 3. Aktiver App Store-produktet
Aktiver produktet **"App Store"** under tjenester.
### 4. Tildel riktige roller
1. **Gå til** "Brukere og Roller"
2. **Finn brukeren** som skal kunne aktivere integrasjoner
3. **Tildel rollen** "Integration Administrator"

### 5. Verifiser tilgang
Brukeren kan nå:
- **Logge seg inn** via [apps.visma.com](https://apps.visma.com)
- **Se App Store** som eget menyvalg neste gang de logger inn i Visma Home

## Visma App Store tilgjengelig i Visma Home
Når aktiveringen er fullført, vil App Store være synlig som et eget menyvalg i Visma Home.
