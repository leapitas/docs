# Install in Business Central
This document provides an introduction to how you can install Leap Visma Payroll Import in Dynamics 365 Business Central.

## Microsoft AppSource
Leap Visma Payroll Import is available on Microsoft AppSource.

### Installation Steps

1. **Go to Microsoft AppSource** and search for "Leap Visma Payroll Import"
2. **Click on the correct link** in the search results
3. **Select "Free Trial"**
4. **Leap Visma Payroll Import** will now be installed in Business Central
5. **The app is ready** for activation and setup

:::info Login
If you need to log in to AppSource, use your Microsoft 365 user. This is usually the same user you log into Dynamics 365 Business Central with.
:::
