# About Visma Payroll Import

Leap Visma Payroll Import enables transfer of payroll transactions from Visma Payroll to Dynamics 365 Business Central, ensuring accurate and efficient payroll processing.

## Key Features

- **Automatic data transfer**: Import of payroll transactions from Visma Payroll
- **Flexible account mapping**: Customize accounts between systems
- **Dimension support**: Support for transferring dimensions and dimension values
- **Transaction grouping**: Reduce number of transactions by grouping similar entries
- **Error handling**: Robust handling of missing accounts and dimensions
