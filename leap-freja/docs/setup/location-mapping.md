# Location Mapping

Leap Freja requires a mapping between Business Central locations and Freja warehouse locations. Only orders assigned to a mapped location can have Freja assignments created.

## Freja Locations

The **Freja Locations** page lists the available Freja warehouse location codes. These are provided by Freja and represent the physical warehouses managed by Freja on your behalf.

Open the **Freja Setup** page and click **Freja Locations** in the ribbon under **Navigation**.

## Creating Location Mappings

**Freja Location Mappings** link a Business Central location code to a Freja location code.

Open the **Freja Setup** page and click **Freja Location Mappings** in the ribbon under **Navigation**.

1. Click **New** to create a new mapping
2. Select the **Location Code** — the Business Central location used on your orders
3. Select the **Freja Location Code** — the corresponding Freja warehouse code

Once a mapping exists, orders using that Business Central location are eligible for Freja assignment creation.
